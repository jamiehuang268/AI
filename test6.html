🧪 安卓Chrome全能力测试

先解决协议限制，再测试功能

⚠️ 当前协议限制：file:// 你正在通过 file:// 打开此文件。
Chrome会假装"User denied"来阻止摄像头视频流、麦克风、精确定位等API。
解决办法（二选一）：
🚀 方案A：通过局域网IP访问（推荐） 最简单
部署到局域网，手机用IP访问，立即解除所有限制
如果你用电脑：在此文件所在目录打开终端：
python3 -m http.server 8080
如果你只有手机：安装Termux，执行：
pkg install python -y && python -m http.server 8080
然后用手机Chrome访问：
http://你电脑/手机的IP:8080
📲 方案B：添加到主屏幕（PWA） 入口较深
必须是方案A成功后，菜单里才会出现选项。file://下不会出现。
安卓Chrome真实操作路径： 1️⃣ 地址栏右侧查找图标 如果符合PWA标准，地址栏右侧（书签星号旁）会出现 ➕ 或 📥，直接点击。 2️⃣ 如果没有图标，点击 ⋮ 菜单 向下滑动，寻找： • "安装 [应用名称]" • "添加到主屏幕" • "安装应用" 3️⃣ 如果菜单里没有 说明当前是 file:// 协议。请先执行方案A，用IP访问后才会出现。 4️⃣ 国产手机特殊路径 • 小米/红米：菜单 → 工具箱 → 添加到桌面 • 华为：部分机型阉割此功能，建议直接用方案A • OPPO/realme：菜单 → 添加到桌面 • vivo/iQOO：菜单 → 加入桌面 / 工具箱 5️⃣ 安装后从桌面图标启动 必须从桌面新图标进入，不要从历史记录进。
